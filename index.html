Transitions

Positioning

Up 
						
    a.button.trans-bg-top {
    	position: relative;
		transition: top 0.3s ease;
	}

	a.button.trans-bg-top:hover {
		top: -5px;
	}
Slide 
						
    a.button.trans-bg-slide {
		left: 0;
		position: relative;
		transition: left 0.6s ease;
	}

	a.button.trans-bg-slide:hover {
		left: 30px;
	}
